# User Stories (MoSCoW)

## MUST
- As an Admin I can create workflows with ordered steps.
- As a Worker I can start a task and be guided step-by-step.
- As a Trainer I can attach a video to each step.
- As a Manager I can view task completion and progress.

## SHOULD
- Support QR/barcode scanning validation
- Provide role-based access controls
- Track time per step

## COULD
- Add multilingual support
- Add voice guided instructions

## WON'T
- Replace ERP systems in MVP
- Include accounting features
